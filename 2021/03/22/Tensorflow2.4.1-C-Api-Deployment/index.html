简单介绍

作者最近打算在服务器上部署用python训练的tensorflow模型，同时使用C++Api调用这个模型。本篇文章主要是记录如何编译和简单使用tensorflowC++api的动态链接库。

1.环境

Ubuntu18.04

Python3.8.5

Anaconda3

tensorflow 2.4.1（gpu）

Bazel 3.1.0

protobuf 3.9.2

cuda 11.0

cudnn 8.0

gcc7.5

nvidia Titan
